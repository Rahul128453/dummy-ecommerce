# Dummy E-Commerce Application

## Project Overview

This project is a **dummy e-commerce application** built mainly to demonstrate **frontend development concepts and functionality implementation**.

The main focus of this project is on **application logic, feature development, and learning purposes**, rather than UI/UX design.

## Important Note

* This project **does not focus on design, styling, or responsiveness**.
* The UI is kept **simple and basic** intentionally.
* The purpose of the project is to **practice development features and functionality**.

## Key Features

* Product listing
* Add / delete products (Admin Dashboard)
* Wishlist functionality
* Cart functionality
* Order placement
* Order history
* Real-time order updates using Socket functionality

## Technologies Used

* React.js
* TypeScript
* Tailwind CSS
* JSON Server (Mock API)
* Socket functionality for real-time events

## Disclaimer

This project is created **only for practice and learning purposes**.
It is not intended to be a production-ready or fully designed application.

The main goal of this project is to **focus on development logic rather than design or responsiveness**.
